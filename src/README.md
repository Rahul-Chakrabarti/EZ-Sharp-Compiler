EZ Sharp is a new coding language developed fully in C.  This language has 4 different phases: Lexical, Syntax, Semantic, and Intermediate Code Generation.  The compiler will go through the sample code (TestSyn.cp) and execute all of the phases.  If the code passes all of the tests, it is grammatically correct based off of our language!  If there are any errors, the type of error and the location will be displayed in error.txt.
